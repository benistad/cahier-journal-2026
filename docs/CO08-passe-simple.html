CO.08 – Le passé simple

Le passé simple est un temps du passé utilisé principalement à l'écrit (récits, contes, romans, textes historiques). On le trouve surtout dans les livres.

Les verbes du 1er groupe au passé simple

Pour conjuguer un verbe du 1er groupe au passé simple, on prend le radical du verbe (infinitif sans -er) et on ajoute les terminaisons :

-ai   -as   -a   -âmes   -âtes   -èrent

Exemple avec le verbe chanter :

Je chantai nous chantâmes Tu chantas vous chantâtes il/elle chanta ils/elles chantèrent

Ex : Le roi régna pendant vingt ans.

Ex : Les enfants jouèrent dans le jardin.
